# Selenium Web Automation with Page Object Pattern

A basic web test that created with the Page Object Pattern. To start test build & run mainTest.java file.


## Dependencies & Tools

* Selenium (ver. 3.141.5)

* Log4J (ver. 2.14.1)

* JUnit (ver. 4.13.2) (.jar)

* Maven
